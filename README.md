# Analysis of De Novo Mutations from Trios-based Whole-Genome Sequencing

This repository contains R scripts for analyzing de novo mutations (DNMs) based on whole-genome sequencing data, with a focus on parental factors, assisted reproductive technologies (ART), and offspring outcomes.

---

## Overview

All analyses are conducted using multivariable regression models in R, with appropriate adjustment for predefined covariates.

- Linear regression: Beta (95% CI)  
- Logistic regression: Odds Ratio (OR, 95% CI)
